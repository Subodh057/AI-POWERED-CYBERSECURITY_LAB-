# 🛡️ AI-Powered Splunk Log Analyzer

An AI-driven cybersecurity project that integrates **Splunk logs with Generative AI** to automatically analyze security events and generate actionable insights.

---

## 🚀 Overview

This project simulates a **Security Operations Center (SOC) workflow** by:

- Fetching logs directly from **Splunk via REST API**
- Monitoring log updates using **Watchdog (real-time detection)**
- Processing and structuring log data
- Using **Google Generative AI (Gemini)** to analyze threats
- Generating:
  - 📄 Human-readable reports
  - 🚨 Alert summaries with priority levels

---

## ⚙️ Tech Stack

- 🐍 Python
- 📊 Splunk REST API
- 👀 Watchdog (real-time file monitoring)
- 🤖 Google Generative AI (Gemini)
- 📦 Pandas
- 🔐 python-dotenv
