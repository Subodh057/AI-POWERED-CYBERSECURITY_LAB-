🛡️ AI-Powered Network Traffic Analyzer (Scapy-Based)

An AI-driven cybersecurity project that performs real-time network traffic monitoring, detects suspicious behavior, and generates SOC-style alerts and reports.

⸻

🚀 Overview

This project simulates a Security Operations Center (SOC) workflow by:

* Capturing live network traffic using Scapy
* Detecting suspicious behavior (e.g., port scanning)
* Applying rule-based detection logic
* Using Google Generative AI (Gemini) to analyze threats
* Generating:
    * 📄 Human-readable reports
    * 🚨 Structured alert outputs

⸻

❓ Why Not Wireshark?

Wireshark is a powerful GUI-based packet analysis tool, but it is primarily used for manual inspection.

This project uses Scapy instead of Wireshark because:

* Enables automation of packet capture and analysis
* Allows custom detection logic in Python
* Supports real-time monitoring pipelines
* Easily integrates with AI for automated threat explanation
